# Javascript-brower-object-module-bom

A beginner-friendly JavaScript project demonstrating the concepts of the **Browser Object Model (BOM)**. This repository contains practical examples of how JavaScript interacts with the web browser using the `window` object and its related properties and methods. The Browser Object Model allows JavaScript to control browser behavior, navigate pages, access browser information, and display dialogs. :contentReference[oaicite:0]{index=0}

## 📚 Topics Covered

- Window Object
- Alert, Confirm & Prompt Dialog Boxes
- `window.location`
- `window.history`
- `window.navigator`
- `window.screen`
- Browser Window Size
- `setTimeout()`
- `setInterval()`
- Opening and Closing Browser Windows
- Redirecting Web Pages

## 📖 What is BOM?

The **Browser Object Model (BOM)** enables JavaScript to communicate with the browser itself. Through the `window` object, developers can interact with browser features such as navigation, history, screen information, dialogs, timers, and page location. :contentReference[oaicite:1]{index=1}
